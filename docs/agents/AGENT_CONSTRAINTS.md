# Agent Constraints

## Limitations

### Code Changes
- Must follow coding standards
- Must maintain test coverage >80%
- Must update documentation
- Must pass all tests

### Security
- Cannot commit secrets
- Must follow security guidelines
- Must validate inputs
- Must handle errors properly

### Performance
- Must not introduce performance regressions
- Must optimize database queries
- Must use async where appropriate
- Must cache appropriately

## Required Approvals

- Human review for breaking changes
- Security review for sensitive changes
- Architecture review for major changes
